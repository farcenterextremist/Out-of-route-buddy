## Maintenance

- **Light tier:** Brainstorm; add ideas to BRAINSTORM_AND_TASKS; File Organizer promotes to this list.
- **Medium tier:** **Improve sandboxed ideas each loop.** Add design brief, validation checklist, or advance % for 1–2 ideas. See [SANDBOX_COMPLETION_PERCENTAGE.md](./SANDBOX_COMPLETION_PERCENTAGE.md). Merging should not be taken lightly; update % only when criteria are met.
- **Cap:** When count reaches 50, prioritize or archive before adding. Prune promoted/deprioritized items.
- **Each loop:** File Organizer adds at least 1 Heavy idea when below cap. Medium tier advances % for 1–2 ideas. See [IMPROVEMENT_LOOP_TEAMS.md](./IMPROVEMENT_LOOP_TEAMS.md).
