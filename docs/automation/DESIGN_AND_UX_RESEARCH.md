## Research Topics (Rotate)

| Topic | What to research | Apply in Phase 3 |
|-------|------------------|------------------|
| **Color schemes** | Material Design 3 palettes, contrast ratios, accessibility | One color tweak for contrast |
| **Typography hierarchy** | Font sizes, weights, line height for readability | One text style adjustment |
| **Spacing (8dp grid)** | Material spacing scale, padding, margins | 1–2dp spacing on 8dp grid |
| **Elevation** | Shadow, elevation levels for cards and surfaces | Verify 6dp elevation per UI_CONSISTENCY |
| **State flows** | Loading, error, empty states; skeleton screens | Improve one state flow |
| **Fleet/driver app patterns** | Professional driver apps, trucking UI conventions | One pattern alignment |
| **Beautification standards** | Spacing, typography, elevation, professional UI | One subtle improvement |

## Advanced (UI/UX Focus)

When UI/UX is this run's focus, extend research to 15 min:

- **Color theory** — Complementary, analogous, triadic; contrast for accessibility
- **Typography hierarchy** — Heading scale, body text, captions
- **Material Design 3** — Compare current app to MD3 guidelines
- **Fleet/driver app patterns** — Professional fleet management UI; driver-facing dashboards
