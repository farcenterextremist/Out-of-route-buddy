# Subject-line templates (Email Editor / Market Guru)

**For:** Human-in-the-Loop Manager — pick one that fits the email purpose.  
**Created:** 2025-02-19  
**References:** `docs/agents/COMPREHENSIVE_AGENT_TODOS.md` (Email Editor section), `docs/agents/OPEN_LINE_OF_COMMUNICATION.md`

---

## 1. We need your decision

Use when the coordinator or team needs the user to choose, prioritize, or decide something (e.g. feature order, scope, or a yes/no).

| Example | When to use |
|--------|-------------|
| **OutOfRouteBuddy — need your decision** | Generic; works for any decision ask. |
| **OutOfRouteBuddy — which should we do first?** | When asking for priority (e.g. Auto drive vs Reports vs History). |
| **Decision needed: [topic]** | Replace `[topic]` with a short label (e.g. "Statistics change", "Next feature"). |

---

## 2. Here's an update

Use when sending status, progress, or informational updates (no reply required, but welcome).

| Example | When to use |
|--------|-------------|
| **OutOfRouteBuddy — quick update** | Short status; user can skim. |
| **OutOfRouteBuddy — this week: [X]** | Replace `[X]` with a one-line summary (e.g. "Statistics monthly only, todos updated"). |
| **Update: [brief description]** | Replace with e.g. "Roadmap and comprehensive todos are in place — review when you can." |

---

## 3. Please confirm

Use when you need the user to confirm receipt, approval, or a simple yes/no (e.g. "Is this OK?", "Can we proceed?").

| Example | When to use |
|--------|-------------|
| **OutOfRouteBuddy — please confirm** | Generic confirmation ask. |
| **Please confirm: [what]** | Replace `[what]` (e.g. "Statistics monthly only", "Next sprint scope"). |
| **Quick confirm? [one-line question]** | E.g. "Quick confirm? Proceed with Auto drive as #1?" |

## Value proposition (one sentence)

For app store listings, outreach, or elevator pitch:

**OutOfRouteBuddy helps you track trips and out-of-route miles simply and reliably—so you always know where you drove and how much was OOR.**

*(Optional variants: "...so you have a clear record for reporting." / "...so you spend less time on paperwork and more on the road.")*

---

## Suggested sign-off for coordinator emails

Use a consistent sign-off so the user knows the message is from the team, not an individual bot.

**Recommended:**

- **— OutOfRouteBuddy Team**

**Alternatives (pick one and stick with it):**

- **— The OutOfRouteBuddy Team**
- **— OutOfRouteBuddy** *(shorter)*
- **— Coordinator, OutOfRouteBuddy** *(if you want to signal the coordinator role explicitly)*

Keep it short and professional; "— OutOfRouteBuddy Team" is clear and brand-aligned.

---

## Tone (Board-adopted)

- **Professional, short, one clear ask per email** so the user can reply without decoding jargon.
- One main question or request per message when possible; use bullet points for multiple items.
