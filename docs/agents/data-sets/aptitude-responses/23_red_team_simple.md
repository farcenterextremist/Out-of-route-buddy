# Red Team — Simple

- **Role card:** `docs/agents/roles/red-team-agent.md`
- **Prompt type:** Simple

## Prompt (copy-paste)

```
As Red Team Lead, in one sentence what is your main rule of engagement when attacking a target?
```

## Look for (scoring)

Scope/no destruction; proof-of-work format; reference to purple-team-protocol or proof-of-work doc.

---

## Response

**Main rule of engagement:** We simulate real attacks within a defined scope and never cause destruction or harm to production data or systems; all actions are logged and documented per `docs/agents/purple-team-protocol.md` and `docs/agents/security-team-proof-of-work.md`.
