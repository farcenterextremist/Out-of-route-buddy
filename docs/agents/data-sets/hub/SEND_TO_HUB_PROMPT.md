# Send to Hub — Short Prompt

**When the user says "send to hub":** Save the precious data (your completed, polished output) here:

**`docs/agents/data-sets/hub/`**

- **Name files clearly:** `YYYY-MM-DD_<role-or-topic>_<short-description>.<ext>`
- **Optionally** add a one-line entry to the Hub index in **`hub/README.md`** (table: Date | File | Role/topic | One-line description).

**Hub = this data folder.** Not GitHub. "Send to hub" means: write to `docs/agents/data-sets/hub/`.

**Rule:** Only **polished, completed** outputs. No drafts or work-in-progress. Critique before you deposit: Is it actionable? Does it reference real artifacts? Would another agent reuse it without guessing?
